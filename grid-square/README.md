# Grid Square Project

This project demonstrates how to create a simple square layout using CSS Grid. It consists of an HTML file and a CSS file that styles the items in a grid format.

## Project Structure

```
grid-square
├── src
│   ├── index.html
│   └── css
│       └── styles.css
├── .gitignore
└── README.md
```

## Getting Started

To set up this project, follow these steps:

1. **Clone the repository** (if applicable):
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd grid-square
   ```

3. **Open the `src/index.html` file** in your web browser to view the grid layout.

## Usage

The `index.html` file contains a simple structure with a container and several items. The `styles.css` file applies grid styles to create a square layout.

## Customization

Feel free to modify the number of items or their styles in the `styles.css` file to create different layouts or designs.
